Native platform components

Remero release builds also bundle native Android and iOS libraries (for example Firebase, Google Play services, AndroidX, and store billing SDKs) through Gradle and CocoaPods. Those components are not enumerated in pubspec.lock.

Needs verification: a complete native third-party license list should be generated from a release build artifact (for example Android third_party_licenses.txt / merged NOTICES) before final store submission.

Needs verification

  • Native Android/iOS SDK inventory and license texts for the exact Remero 1.0.0+23 release APK/IPA.
  • Any license type marked above if the upstream package changes its LICENSE file in a future lockfile revision.

네이티브 플랫폼 구성요소

Remero 릴리스 빌드에는 Gradle/CocoaPods를 통해 Android/iOS 네이티브 라이브러리(Firebase, Google Play services, AndroidX, 스토어 결제 SDK 등)도 포함됩니다. 이들은 pubspec.lock에 나열되지 않습니다.

Needs verification: 최종 스토어 제출 전 릴리스 빌드 산출물(예: Android third_party_licenses.txt, 병합 NOTICES)에서 네이티브 제3자 라이선스 전체 목록을 생성·검증해야 합니다.

Needs verification

  • Remero 1.0.0+23 릴리스 APK/IPA에 포함된 네이티브 Android/iOS SDK 목록 및 라이선스 전문.
  • 향후 lockfile 변경으로 upstream LICENSE 파일이 바뀌는 경우 해당 패키지의 라이선스 유형.
